スマート履歴書ジェネレーター

履歴書の入力、写真配置、志望動機・自己PRの作成、印刷・PDF保存をブラウザで行える履歴書作成支援ツールです。

最新UIでは、全画面表示、プレビュー拡大、基本情報の保存 / 読込 / 削除に対応しました。BOOTH / Vector / ダウンロード導線は引き続き準備中です。

Overview

履歴書づくりを、ひとつの画面で見やすく整える。

スマート履歴書ジェネレーターは、日本向けの履歴書をブラウザ上で整え、入力から印刷・PDF保存までを一つの流れで進められる支援ツールです。

JIS標準、転職用、アルバイト用のテンプレートを切り替えながら、基本情報、学歴、職歴、資格、志望動機、自己PRなどを編集できます。最新UIでは広い画面を使いやすく整理し、写真配置や印刷前プレビューも見やすく確認できる構成に整えています。

Features

主な機能

書

履歴書テンプレート切替

JIS標準、転職用、アルバイト用のテンプレートを切り替えて利用できます。

広

全画面表示とプレビュー拡大

広い画面で入力とプレビューを見比べやすくし、履歴書の仕上がりを大きく確認できます。

文

文章入力とAI補助

志望動機・自己PRを手動入力でき、Gemini API を使った文章生成にも対応しています。AIアシスタントUI自体は引き続き利用できます。

入力・編集機能

  • JIS標準 / 転職用 / アルバイト用テンプレート切替
  • フォント切替
  • ふりがな形式切替
  • 学歴 / 職歴 / 資格の動的追加
  • 志望動機 / 自己PR入力
  • 基本情報の保存 / 読込 / 削除
  • localStorage保存

出力・補助機能

  • Ban.Tai ロゴ表示
  • 全体デザイン整理
  • 印刷 / PDF保存
  • 写真アップロードとトリミング
  • Gemini API を使ったAI文章生成
  • APIキー未設定時のテンプレート文フォールバック
Highlights

特色

  • ブラウザ中心で扱えるため、軽量で導入しやすい構成です。
  • 全画面表示ボタンとプレビュー拡大表示により、仕上がり確認をより大きく行えます。
  • 氏名、住所、連絡先などの基本情報をプロフィールとして保存し、入力の繰り返しを減らせます。
  • スマホ幅でも横スクロールが出ないことを確認済みです。

AI機能について

AI文章生成を使う場合は、利用者自身のGemini APIキー設定が必要になる場合があります。履歴書には個人情報が含まれるため、AI機能に入力する内容は利用者自身で確認してください。

なお、AIアシスタントUIは維持していますが、外部API実行によるAI機能本体の確認は、現時点ではまだ未実施です。

個人情報についての注意

入力内容は基本的に利用者のブラウザ内で扱う設計ですが、AI機能を使う場合は外部APIとの通信が発生する可能性があります。公開前に、この点の説明をさらに整備する必要があります。
