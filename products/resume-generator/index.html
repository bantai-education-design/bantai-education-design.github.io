Overview

履歴書づくりを、ひとつの画面で見やすく整える。

スマート履歴書ジェネレーターは、写真付き履歴書をかんたんに作成できるWindows用アプリです。氏名・住所・生年月日などの基本情報から、学歴、職歴、免許・資格、志望動機、自己PR、本人希望欄まで、履歴書に必要な項目を順番に入力できます。

画面右側には履歴書のプレビューが表示されるため、仕上がりを確認しながら編集できます。証明写真画像の読み込みにも対応しており、JPG / PNG の写真を読み込み、履歴書の写真枠に合わせて位置や大きさを調整できます。

作成した履歴書は、表面・裏面を正しい向きの2ページPDFとして出力できます。実際に両面印刷を行う場合は、作成されたPDFを開き、PDFビューアーまたはプリンター側の印刷画面で「両面印刷」を選択してください。プリンターによっては、プリンタードライバー側で両面印刷設定が必要になる場合があります。

Highlights

特色

  • 対応OS は Windows 10 / 11。exe を起動して利用します。
  • 表面・裏面を正しい向きの2ページPDFとして作成でき、片面ずつの印刷にも対応します。
  • 氏名、住所、連絡先などの基本情報をプロフィールとして保存し、入力の繰り返しを減らせます。
  • 履歴書の入力・写真配置・保存・PDF作成・印刷はオフラインで利用できます（AI文章作成のみインターネットとGemini APIキーが必要）。

ライセンスと SAMPLE について

10日間の試用に対応しています。試用中・未認証時は、印刷物・PDFの表面・裏面に SAMPLE 透かしが入ります。ライセンスキーを入力すると SAMPLE 表示が解除されます。

BOOTHではアプリ本体を試用版として公開し、ライセンスキー購入により製品利用できる形にします。Vectorは現在登録申請中のため、外部リンクはまだ掲載していません。

AI機能・個人情報についての注意

AI文章生成を使う場合は、利用者自身の Gemini API キー設定が必要になる場合があります。履歴書には個人情報が含まれるため、AI機能に入力する内容は利用者自身で確認してください。AIを使わない通常機能はオフラインで利用できます。
