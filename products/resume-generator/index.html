Smart Resume Generator

スマート履歴書作成ツール

氏名・住所・学歴・職歴・免許資格・志望動機・自己PR・本人希望欄を入力し、写真付き履歴書を作成できるWindows用ツールです。表面・裏面を正しい向きの2ページPDFとして作成できます。

対応OS：Windows 10 / 11。10日間の試用に対応し、未認証時はPDF・印刷物に SAMPLE が入ります。BOOTH / Vector / ダウンロード導線は近日公開です。

Overview

履歴書づくりを、ひとつの画面で見やすく整える。

スマート履歴書作成ツールは、日本向けの履歴書を整え、入力から印刷・PDF保存までを一つの流れで進められるWindows用ツールです。JIS標準、転職用、アルバイト用のテンプレートを切り替えながら、基本情報、学歴、職歴、資格、志望動機、自己PRなどを編集できます。

履歴書は表面（1ページ目）と裏面（2ページ目）を正しい向きの2ページPDFとして作成できます。実際に用紙の両面へ印刷するかどうかは、お使いのプリンターやPDFビューアーの印刷設定によって決まります。両面印刷したい場合は、作成したPDFを開き、印刷画面で「両面印刷」を選んでください。プリンターによっては、プリンタードライバー側で両面印刷を有効にする必要があります。

Features

主な機能

書

履歴書テンプレート切替

JIS標準、転職用、アルバイト用のテンプレートを切り替えて利用できます。

刷

表裏2ページPDFの作成

表面・裏面を正しい向きの2ページPDFとして作成できます。両面印刷はPDFを開いてPDFビューアーまたはプリンタードライバー側で設定します。

文

文章入力とAI補助

志望動機・自己PRを手動入力でき、Gemini API を使った文章生成にも対応しています。AIアシスタントUI自体は引き続き利用できます。

入力・編集機能

  • JIS標準 / 転職用 / アルバイト用テンプレート切替
  • フォント切替 / ふりがな形式切替
  • 学歴 / 職歴 / 資格の動的追加
  • 志望動機 / 自己PR / 本人希望欄の入力
  • 基本情報の保存 / 読込 / 削除
  • localStorage保存

出力・補助機能

  • 表面・裏面を正しい向きの2ページPDFとして作成
  • 表面だけ / 裏面だけ / 表裏の印刷・PDF保存
  • 両面印刷はPDFビューアー / プリンタードライバー側で設定
  • 写真アップロードとトリミング
  • Gemini API を使ったAI文章生成（APIキー未設定時はテンプレート文）
  • 10日間試用・ライセンスキーで SAMPLE 解除
Highlights

特色

  • 対応OS は Windows 10 / 11。exe を起動して利用します。
  • 表面・裏面を正しい向きの2ページPDFとして作成でき、片面ずつの印刷にも対応します。
  • 氏名、住所、連絡先などの基本情報をプロフィールとして保存し、入力の繰り返しを減らせます。
  • 履歴書の入力・写真配置・保存・PDF作成・印刷はオフラインで利用できます（AI文章作成のみインターネットとGemini APIキーが必要）。

ライセンスと SAMPLE について

10日間の試用に対応しています。試用中・未認証時は、印刷物・PDFの表面・裏面に SAMPLE 透かしが入ります。ライセンスキーを入力すると SAMPLE 表示が解除されます。

BOOTH版はアプリ本体を無料配布し、ライセンスキーを別売する予定です（現時点は単一共有キー方式）。Vector版はライセンスキー同梱の製品版を予定しています。

AI機能・個人情報についての注意

AI文章生成を使う場合は、利用者自身の Gemini API キー設定が必要になる場合があります。履歴書には個人情報が含まれるため、AI機能に入力する内容は利用者自身で確認してください。AIを使わない通常機能はオフラインで利用できます。
