学級名簿メーカー

名簿から提出物チェック表まで、
学級事務で繰り返し使う一覧表を一つのツールで。

児童名簿をもとに、通常の学級名簿、提出物チェック表、 集金確認表、誕生日順一覧などを作成できます。

出席番号順、男女別、五十音順、誕生日順への並べ替え、 チェック欄数、ふりがな、書体、行間、余白も調整できます。

Features

学級で使う名簿を、目的に合わせて整える

児童名簿を一度用意すれば、並べ方や表示項目、チェック欄を変更しながら、
用途の異なる一覧表を作成できます。

01

名簿の並べ替え

  • 出席番号順
  • 男女別
  • 五十音順
  • 誕生日順
  • 表示しない児童の一時除外

用途に合わせた並び順へ、名簿を書き直さず切り替えられます。

02

提出物チェック表

  • チェック欄数を0列から15列まで調整
  • 表のタイトルを自由入力
  • 行の高さを調整
  • 1段組み／2段組み

宿題、集金、配布物など、確認項目に合わせたチェック表を作成できます。

03

表示と文字の調整

  • 明朝体／ゴシック体
  • ふりがなの表示／非表示
  • 行間の調整
  • 用紙周囲の余白調整

低学年用の読みやすい名簿から、職員用の簡潔な名簿まで整えられます。

04

印刷プレビュー

  • 作成内容をその場で確認
  • 用紙上の完成イメージを表示
  • 印刷時は操作パネルを非表示
  • 表の途中で行が分断されにくい印刷設定

画面で確認した名簿を、そのまま紙へ印刷できます。

Documents

一つの児童名簿から作成できるもの

日々の学級事務で繰り返し使う一覧表を、同じ児童名簿から用途別に整えられます。

  • 通常の学級名簿
  • 提出物チェック表
  • 集金確認表
  • 配布物確認表
  • 誕生日順一覧
  • 五十音順一覧
  • 男女別一覧
  • ふりがな付き名簿

How To

基本の使い方

  1. 01名簿データを確認する
  2. 02並べ替えと表示方法を選ぶ
  3. 03チェック欄・書体・余白を調整する
  4. 04プレビューを確認して印刷する
