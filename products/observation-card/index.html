Picture Diary & Observation Card Maker

絵日記・観察カード作成メーカー

小学生の「見たこと」「気づいたこと」「考えたこと」を、紙に残しやすくする印刷用紙作成ツールです。

Windows EXE版 v1.2.0 をBOOTHで公開中です。10日間無料で試用できます。

BOOTHで購入・ダウンロードする 配布ページを見る 注意事項
Overview

絵の欄と文章欄を組み合わせた、観察カードをすぐに作成。

Ban.Tai 絵日記・観察カード作成メーカーは、小学校低学年、生活科、理科、夏休みの自由研究、家庭学習で使える観察カードや絵日記用紙を作成するためのWindows用アプリです。

アプリを起動し、用紙サイズ、絵の広さ、マス目、横罫線、縦罫線、余白、線の色、線の濃さを選んで印刷できます。PDF保存にも対応しているため、授業用プリントや家庭学習用の用紙を事前に準備しやすくなります。

10日間無料で試用できます。試用中の印刷/PDF保存には「SAMPLE / サンプル」の透かしが入り、ライセンスキー（480円）の認証で透かしと期限が解除されます。

For Whom

対象

大きめのマス目や絵の欄で、初めての記録活動を支えます。

植物、生きもの、天気、実験などの観察を絵と文章で残せます。

夏休みの自由研究や家庭での絵日記用紙づくりに使えます。

Features

主な機能

紙

用紙サイズ設定

A4、B4、八つ切、四つ切の用紙設定に対応しています。

絵

絵と文章の欄

絵を描く欄と、文章を書く欄を組み合わせたカードを作れます。

線

マス目・罫線

マス目、横罫線、縦罫線を切り替え、サイズや行数も調整できます。

How To Use

使い方

  1. ZIPを展開し、インストーラーEXEを実行してアプリを起動します。
  2. 用紙サイズ、絵の広さ、文章欄のスタイルを選びます。
  3. 必要に応じて余白、線の色、線の濃さを調整します。
  4. 「PDF出力 / 印刷」ボタンから印刷またはPDF保存を行います。

注意書き

  • Windows用EXE版はコード署名を行っていないため、初回起動時にWindows SmartScreen等の確認画面が表示される場合があります。
  • EXEを実行できない環境向けに、HTML単体版を保険として同梱しています。
  • 未認証・試用中の印刷/PDF保存には「SAMPLE / サンプル」の透かしが入ります。
  • オフライン環境では、外部CDNに依存する表示の一部が変わる可能性があります。
  • 印刷結果はブラウザやプリンター設定により変わる場合があります。
  • GitHubは開発管理用であり、一般利用者向けの直接配布先にはしません。
