Picture Diary & Observation Card Maker

絵日記・観察カード作成メーカー

小学生の「見たこと」「気づいたこと」「考えたこと」を、紙に残しやすくする印刷用紙作成ツールです。

HTML単体版 v1.0.2。BOOTH / Vectorでの公開準備を進めています。

配布予定を見る 注意事項
Overview

絵の欄と文章欄を組み合わせた、観察カードをすぐに作成。

Ban.Tai 絵日記・観察カード作成メーカーは、小学校低学年、生活科、理科、夏休みの自由研究、家庭学習で使える観察カードや絵日記用紙を作成するためのHTML単体アプリです。

ブラウザでHTMLファイルを開き、用紙サイズ、絵の広さ、マス目、横罫線、縦罫線、余白、線の色、線の濃さを選んで印刷できます。PDF保存にも対応しているため、授業用プリントや家庭学習用の用紙を事前に準備しやすくなります。

スクリーンショット準備中

画像なしでも崩れない掲載構成

正式なスクリーンショットができるまでは、用紙イメージの仮表示で掲載します。画像リンク切れが起きないよう、存在しない画像は参照していません。

For Whom

対象

小学校低学年

大きめのマス目や絵の欄で、初めての記録活動を支えます。

生活科・理科

植物、生きもの、天気、実験などの観察を絵と文章で残せます。

自由研究・家庭学習

夏休みの自由研究や家庭での絵日記用紙づくりに使えます。

Features

主な機能

紙

用紙サイズ設定

A4、B4、八つ切、四つ切の用紙設定に対応しています。

絵

絵と文章の欄

絵を描く欄と、文章を書く欄を組み合わせたカードを作れます。

線

マス目・罫線

マス目、横罫線、縦罫線を切り替え、サイズや行数も調整できます。

How To Use

使い方

  1. ZIPを展開し、HTMLファイルをブラウザで開きます。
  2. 用紙サイズ、絵の広さ、文章欄のスタイルを選びます。
  3. 必要に応じて余白、線の色、線の濃さを調整します。
  4. 「PDF出力 / 印刷」ボタンから印刷またはPDF保存を行います。
