# 絵日記・観察カード作成メーカー 紹介動画構成案

## 動画タイトル案

絵日記・観察カード作成メーカーの使い方

## 長さ

30秒〜60秒程度。

## 画面構成

1. タイトル画面
   - 絵日記・観察カード作成メーカー
   - HTML単体版 v1.0.2
   - A4印刷・PDF保存対応

2. 起動
   - ZIPを展開する
   - `絵日記・観察カード作成メーカー.html` をブラウザで開く

3. 用紙確認
   - 絵の欄と文章欄があることを見せる
   - 生活科、理科、自由研究、家庭学習に使えることを短く示す

4. 設定
   - 用紙サイズ、マス目、横罫線、縦罫線、余白などを選べることを見せる

5. 印刷・PDF保存
   - 「PDF出力 / 印刷」ボタンを押す
   - 印刷画面でPDF保存またはプリンター印刷を選ぶ

6. まとめ
   - 絵と文章で観察を残せる
   - 公式HP、BOOTH、Vectorへの案内

## 入れる文字

- 小学校低学年の絵日記に
- 生活科・理科の観察カードに
- 夏休みの自由研究に
- ブラウザで開いてすぐ使える
- A4印刷・PDF保存対応
- HTML単体版 v1.0.2

「絵日記・観察カード作成メーカーは、小学校低学年の絵日記や、生活科・理科の観察記録に使えるA4印刷用ツールです。ZIPを展開してHTMLファイルをブラウザで開くだけで、絵の欄と文章欄を組み合わせた観察カードを作成できます。用紙を確認したら、PDF出力・印刷ボタンから保存や印刷を行います。授業準備、家庭学習、自由研究にご活用ください。」

## HPに埋め込む場合の注意点

- GitHub Pagesで表示できる静的なMP4にする。
- 動画ファイル候補は `assets/videos/observation-card-demo.mp4`。
- ポスター画像候補は `assets/images/observation-card/video-poster.png`。
- 実ファイルを配置するまでは、HTMLに動画や画像の参照を入れない。
- MP4は容量を抑え、スマホ回線でも読み込みやすくする。
- 自動再生は使わず、利用者が再生ボタンを押せる構成にする。
- 音声なしでも内容が伝わるよう、画面内テキストを入れる。
