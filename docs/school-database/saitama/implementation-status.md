# 埼玉県学校宛先データベース 実装状況

更新日: 2026-07-22

## 目的

東京都学校宛先データベースの機能と操作感を維持しながら、埼玉県内の公立・私立の幼稚園、小学校、中学校、義務教育学校、高等学校、中等教育学校、特別支援学校を検索できるページを追加する。

## 実装ブランチ

`feature/saitama-school-database`

## 現行東京都版の確認結果

- ページ: `tools/tokyo-school-address/index.html`
- 検索処理: `tools/tokyo-school-address/search.js`
- スタイル: `tools/tokyo-school-address/style.css`
- データ: `data/tokyo_public_schools_address_2025.json`
- 現行データ項目: `school_name`, `school_name_kana`, `school_type`, `establishment_type`, `municipality`, `postal_code`, `address`, `phone`
- 東京都固有処理: データURL、自治体表示順、ページ文言、収録件数表

## データ年度の判断

令和8年度埼玉県学校便覧は、2026年7月時点では学校名・校長名等のみの速報版を7月、完成版を8月下旬に公開予定と案内されている。住所・電話番号を含む検索データの初期作成は、令和7年度学校便覧の完成版を基準にする。

公開前に令和8年度完成版が公開された場合は、同じ変換処理で差し替えて再検証する。

## 実装順

1. 公式原本のURLと基準日を記録する。
2. Excelの列構造を校種別に調査する。
3. 埼玉県データを東京都版互換JSONへ変換する。
4. 重複、欠損、郵便番号、電話番号、自治体名を検証する。
5. 埼玉県検索ページを新設する。
6. 埼玉県版で共通検索エンジンを検証する。
7. 安定後に東京都版の共通化を行う。
8. 全国学校データベース入口を追加する。

## 初期版で扱わない項目

- 校長名
- 個人メールアドレス
- 児童生徒数
- 学級数
- 学校写真
- 認定こども園
- 保育所
- 大学、専修学校、各種学校

## 安全基準

- 学校名、住所、電話番号は推測補完しない。
- 原本にない情報は空欄にする。
- 休校、廃校、分校、分教室、併置課程は個別に確認する。
- 東京都版3,509校・園の表示と検索を回帰テストする。
