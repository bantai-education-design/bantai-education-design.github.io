## 令和8年度への更新

令和8年度学校便覧は、2026年7月に速報版、2026年8月下旬に完成版が公開予定。完成版公開後に次を行う。

1. 令和7年度原本との差分抽出
2. 新設、統合、廃止、移転、電話番号変更の確認
3. JSON再生成
4. 件数再照合
5. 公開ページの基準日更新

## 原本利用ルール

- 原本ファイルは加工前の状態で保存する。
- 加工データとは別フォルダーで管理する。
- ダウンロード日、基準日、元ページ、ファイル名を記録する。
- 学校名、住所、電話番号を推測で補わない。
- 修正が必要な場合は修正理由と確認元を記録する。
