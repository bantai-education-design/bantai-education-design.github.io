埼玉県版

このページは、Ban.Tai Education Design が提供する埼玉県学校宛先データベースです。アプリのインストール不要で、埼玉県の公立・私立学校（全1,929校・園）の宛先検索、宛名コピー、Google Maps表示、CSVダウンロードを利用できます。

💡 本データベースの収録範囲（合計 1,929 校・園）
学校種別 公立 私立 小計
幼稚園 ✓ (33園) ✓ (411園) 444園
小学校 ✓ (781校) ✓ (6校) 787校
中学校・義務教育学校 ✓ (415校) ✓ (32校) 447校
高等学校・中等教育学校 ✓ (137校) ✓ (56校) 193校
特別支援学校 ✓ (57校) ✓ (1校) 58校

※公立校・園は埼玉県教育委員会「令和8年度埼玉県学校便覧」、私立校・園は埼玉県学事課「埼玉県内の私立学校一覧（令和8年4月1日現在）」の公式データをもとに作成しています。

出典について

本データベースの学校情報は、埼玉県教育委員会「令和8年度 埼玉県学校便覧」および埼玉県学事課「埼玉県内の私立学校一覧（令和8年4月1日現在）」の公式データをもとに整理・作成しています。

ご利用上の注意

学校名、住所、電話番号等は変更される場合があります。正式な文書送付や重要な連絡の前には、各学校・自治体の公式情報をご確認ください。本データは元資料の内容を保証するものではありません。
